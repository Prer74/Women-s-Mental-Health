🌸 Women's Wellness Guide

An AI-powered web application that provides personalized mental health support for women using **Machine Learning**, **NLP**, and **real-time communication**. The platform analyzes journal entries, recommends self-care activities, and offers community support in a secure environment.

## ✨ Features

- 📝 Journal sentiment analysis using Logistic Regression
- 🤖 Personalized wellness recommendations using Decision Tree
- 📊 Weekly mood reports and emotional trend visualization
- 💬 Real-time community chat with Socket.IO
- 🔒 Secure journal encryption using Fernet
- 🎵 Self-care resources including music, exercises, and articles
- 🚨 Mental health alerts and support recommendations

## 🛠️ Tech Stack

- Python
- Flask
- HTML, CSS, JavaScript
- MySQL
- Machine Learning (Scikit-learn)
- NLP
- Socket.IO
- Matplotlib
- Cryptography (Fernet)

## 🚀 How It Works

1. User writes a journal entry.
2. Sentiment is analyzed using Machine Learning.
3. Personalized wellness recommendations are generated.
4. Weekly mood reports are created.
5. Users can access community support and self-care resources.
